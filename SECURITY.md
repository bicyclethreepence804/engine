# Security

## Reporting a vulnerability

Please report security issues **privately** so we can assess and patch before public disclosure.

- Prefer **GitHub Security Advisories** on this repository (if enabled).
- Alternatively, email: `kiploksautomation@gmail.com`

Do not open a public issue for an unfixed vulnerability.

## Scope

This policy covers the packages in this repository (`@kiploks/engine-*`). Third-party dependencies are tracked through normal dependency updates.
